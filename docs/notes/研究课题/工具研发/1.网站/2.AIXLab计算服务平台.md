## 系统架构设计

1. 整体架构
```bash
┌─────────────────────────────────────────────────────────────┐
│                       前端 (Vue 3 + Vite)                   │
├─────────────────────────────────────────────────────────────┤
│ 数据管理模块 │ 分析界面模块 │ 结果可视化模块 │ 用户管理界面  │
└──────────────────┬──────────────────────────────────────────┘
                   │ HTTP/REST API
┌──────────────────┼──────────────────────────────────────────┐
│           主后端服务 (Python Flask/FastAPI)                 │
├─────────────────────────────────────────────────────────────┤
│ 用户认证 │ 任务调度 │ 数据管理 │ 服务协调 │ 结果缓存         │
└──────────────────┬──────────────────────────────────────────┘
                   │ 消息队列/HTTP调用
    ┌──────────────┼──────────────┐
    │              │              │
┌───▼────┐   ┌────▼─────┐   ┌───▼─────┐
│Python   │   │R语言分析 │   │MySQL   │
│分析服务  │   │服务      │   │数据库   │
└─────────┘   └──────────┘   └─────────┘
```

### 2. 技术栈细化

- 前端框架: Vue 3 + Composition API + TypeScript

- UI组件库: Element Plus / Ant Design Vue

- 可视化: ECharts / D3.js / Plotly.js

- 状态管理: Pinia

- HTTP客户端: Axios

- 构建工具: Vite

- 主后端: Python FastAPI（轻量、异步支持好）

- Python分析服务: Flask + 科学计算栈（pandas, numpy, scikit-learn等）

- R语言分析服务: plumber（R的Web框架）

- 数据库: MySQL 8.0

- 缓存: Redis（可选，用于会话和结果缓存）

- 任务队列: Celery + Redis（可选，用于长时分析任务）

- 文件存储: 本地文件系统/MinIO对象存储



## AIXLab 前端项目构建代码
### 1. 创建项目
```bash
# 使用 Vite 创建 Vue 3 项目
npm create vite@latest aixlab-frontend -- --template vue

# 进入项目目录
cd aixlab-frontend

# 安装基础依赖(新版的vite已经提示自动安装依赖)
# npm install

# 安装必要的依赖包
npm install vue-router@4 pinia axios element-plus @element-plus/icons-vue
npm install @vueuse/core echarts vue-echarts
npm install monaco-editor @monaco-editor/loader
npm install highlight.js vue3-highlightjs
npm install nprogress
npm install js-cookie
npm install -D sass
```
